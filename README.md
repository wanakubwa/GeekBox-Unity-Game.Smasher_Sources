## O grze

Przy tworzeniu gry zostały wykorzystane następujące technologie:

- Unity - Silnik gry, na którym została w całości stworzona.
- C#    - Język programowania użyty do napisania logiki gry.
- Odin Inspector
- Sirenix Serializer
- DoTween
- More Performance Coroutines
- Easy Mobile Pro
